# 🏨 Royal Stay Hotel

A simple and responsive static hotel website developed using **HTML5** and **CSS3**. The website showcases hotel information, room details, facilities, pricing, and contact information with a clean and modern user interface.

---

## 📌 Project Overview

The **Royal Stay Hotel** website is a front-end mini project designed to demonstrate the fundamentals of web development using HTML and CSS. It provides users with essential information about the hotel through an attractive and easy-to-navigate layout.

---

## ✨ Features

- 🏨 Hotel overview section
- 🛏️ Room details with availability and pricing
- 🏊 Facilities section
- 💰 Pricing information
- 📞 Contact details
- 🔗 Navigation bar with internal links
- 🔝 Back to Top functionality
- 🎨 Modern card-based layout with rounded corners
- 📱 Responsive and user-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
